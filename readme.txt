PROJET 63_51

Pour notre projet dans le cadre du cours 63-51, nous avons utilisé la technologie Hive qui permet d'extraire efficacement des données en provenance de Apache Hadoop. 
Pour ce faire, nous nous sommes basés sur les données : https://github.com/zygmuntz/goodbooks-10k et le tutoriel : https://www.cloudera.com/tutorials/how-to-process-data-with-apache-hive.html. 

Nous avons créé 4 tables : books, ratings, books_tags, tags. Nous nous sommes surtout intéressés aux rating des livres. Nous traitons les livres qui ont les meilleures ratings et les pires ratings. 

Ce git contient les codes que nous avons utilisé pour créer les tables, extraire les données et faire les opérations. Il contient également les images des résultats obtenus.
